You need to have Python3, Cuda, PyTorch, Tensorflow installed in your computer.


For Training the code you need to download the data from https://drive.google.com/file/d/17ZWJOpH1AsYQhoqpWR6PK61HrUhArdAK/view 
and keep the folder data/data and download test images form https://drive.google.com/file/d/1Sz5ZFFZXo3sY85R3v7yJa6W6riDGur46/view
and keep them in the folder data/images and run the train.ipynb.


For Testing the code you can run the test.ipynb, the input is taken from test_images folder by default, however you can change that in the
train.ipynb file and you can get the results in the results folder.
